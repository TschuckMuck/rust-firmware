# Keybaoard Basics

The YAK keyboard is an open source keybaoard based on the famous GH60 Keybaord, so far this is mainly  a playground
to learn something about:

* Keyboards
* ARM Cortex M MCU's
* Embedded Rust
* Low Level Drivers
* An RTOS
  * [Tock Os](https://www.tockos.org/)
